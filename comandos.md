| Comando                                                        | Función                                                    |
|----------------------------------------------------------------|------------------------------------------------------------|
| node -v                                                        | Versión de NodeJS                                          |
| npm install -g @angular/cli                                    | Instalar última versión de Angular                         |
| Set-ExecutionPolicy RemoteSigned -Scope CurrentUser            | Habilitar permisos si no permitió instalar ng              |
| cls                                                            | comando para limpiar consola                               |
| ng version                                                     | Verificar versión Angular                                  |
| git init                                                       | Iniciar repositorio                                        |
| git status                                                     | Consultar el estado de mis archivos                        |
| git add .                                                      | Agregar TODOS los archivos                                 |
| git add .\carpeta                                              | Agregar un archivo en específico                           |
| git commit -m "mensaje"                                        | Comentario sobre lo que se está cargando                   |
| git branch                                                     | Validar la rama en que me encuentro                        |
| git remote add origin https://....                             | Conectar con el repositorio                                |
| git remote -v                                                  | Indica a que repositorio estoy conectada                   |
| git push -u origin main                                        | Subir al repositorio                                       |
| ng new restaurate-proyect --standalone --routing --style=css   | Crear proyecto                                             |
| ng serve -o (ng s -o)                                          | Correr servidor y abrir navegador, carpeta miprimerangular |
| ng generate component view/user/user.component                 | Crear contenido nuevo. Carpeta con html,css,ts,spect.ts    |
| ng g c view/user/user.component                                | (Abrev.) Crear componente de tipo vista                    |
| ng g c view/practica/practica.component                         | nuevo componente para consultar la API                     |
| ng generate service services/user.service/user.service         | Crear un servicio User                                     |
| ng generate service services/practica.service/practica.service | Crear servicio Practica                                    |
| ng generate interface models/user.model/user.model             | Generar un modelo para user                                |
| ng generate interface models/practica.model/practica.model     | Crear modelo para practica                                 |
